== Description ==

Welcome to Aheto Plugin.

Our professional team has designed this page builder to make the site creation process as creative and easy as possible. We are constantly improving it and providing you with tools to create amazing pages for your website.

With the Aheto plugin you can fully customize the website colors, typography, header and footer in a few simple steps. In addition, the plugin is compatible with the 2 most popular WordPress page designers - Elementor and WPBakery.


== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to _Plugins &rarr; Add New.
2. Type "Aheto" into the Search and hit Enter.
3. Locate the Aheto plugin in the list of search results and click **Install Now**.
4. Once installed, click the Activate link.
5. You will be redirected to the Aheto Getting Started page.

It can also be installed manually.

1. Download the Aheto plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Locate Aheto in the list and click the *Activate* link.
5. You will be redirected to the Aheto Getting Started page.
